안내

  1. Egloos 블로그 이사를 쉽게 하기 위해 Hugo를 이용해서 html로 변환합니다.
  2. '미리보기'하면 해당 블로그의 최근 15개 글만 가져옵니다(대기 하지 않고 요청 즉시 가져옵니다)
  3. 1개 글을 다운로드 받는데 2초 정도 걸립니다. 전체 글이 1000개라면 30분 정도 소요됩니다.
  4. 임시저장 완료 시 이메일로 다운로드 주소를 보내드립니다.
  5. 이메일로 받은 압축파일을 해제하신 후 startWeb.bat를 더블클릭해서 실행하신 후 http://localhost:8088을 브라우저로 열어서 확인해주세요
  6. 압축파일을 해제하신 후 public 폴더를 웹서버에 업로드 하시면 이사가 완료 됩니다.
  7. 더 자세한 정보는 여기를 참고하세요
